# LeBron Playoff Performance Analysis

This project analyzes LeBron James’ offensive performance across his career, comparing regular season and playoff statistics. The goal is to identify whether his production changes in high-pressure playoff environments.

## Key Findings
- **Scoring (PPG):** Increased slightly in the playoffs, but not statistically significant.
- **Rebounding (RPG):** Increased in the playoffs and was statistically significant, indicating a consistent boost in rebounding impact.
- **Assists (APG):** Decreased slightly in the playoffs, but not statistically significant.
- **Overall Offensive Production (PRA):** Increased modestly, but did not reach statistical significance.

## Tools Used
- R
- ggplot2
- dplyr
- tidyr
