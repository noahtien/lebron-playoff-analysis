Analyzing Lebron James’ Career Performance in the Regular Season and Playoffs

2026-04-02

Project Overview

This project analyzes LeBron James’ offensive performance across his career, comparing regular season and playoff statistics. The goal is to identify whether his production changes in high-pressure playoff environments.

Data & Tools

  • Data sourced from compiled season statistics from basketball-reference.com
  • Tools used: R, dplyr, tidyr, ggplot2

Performance Differences

To directly compare playoff and regular season performance, I created a differences data frame by subtracting regular season averages from playoff averages for each season. Positive values indicate better playoff performance, while negative values indicate lower production in the playoffs.

# Building the LeBron Playoff and Regular Season Data Frame
lebron_pts_diff <- lebron_combined |>
  group_by(Season,type)|>
  summarize(PTS = mean(PTS, na.rm = TRUE), .groups = "drop")|>
  pivot_wider(names_from = type, values_from = PTS)

lebron_reb_diff <- lebron_combined |>
  group_by(Season,type)|>
  summarize(TRB = mean(TRB, na.rm = TRUE), .groups = "drop")|>
  pivot_wider(names_from = type, values_from = TRB)

lebron_ast_diff <- lebron_combined |>
  group_by(Season,type)|>
  summarize(AST = mean(AST, na.rm = TRUE), .groups = "drop")|>
  pivot_wider(names_from = type, values_from = AST)

# Remove NAs and calculate the difference between playoff and regular season
lebron_pts_diff <- lebron_pts_diff[!is.na(lebron_pts_diff$Playoffs),]
lebron_pts_diff$Playoffs[is.nan(lebron_pts_diff$Playoffs)] <- NA
lebron_pts_diff$pts_difference = lebron_pts_diff$Playoffs - lebron_pts_diff$`Regular Season`

lebron_reb_diff <- lebron_reb_diff[!is.na(lebron_reb_diff$Playoffs),]
lebron_reb_diff$Playoffs[is.nan(lebron_reb_diff$Playoffs)] <- NA
lebron_reb_diff$reb_difference = lebron_reb_diff$Playoffs - lebron_reb_diff$`Regular Season`

lebron_ast_diff <- lebron_ast_diff[!is.na(lebron_ast_diff$Playoffs),]
lebron_ast_diff$Playoffs[is.nan(lebron_ast_diff$Playoffs)] <- NA
lebron_ast_diff$ast_difference = lebron_ast_diff$Playoffs - lebron_ast_diff$`Regular Season`

# Rename the columns
lebron_pts_diff <- lebron_pts_diff |>
  rename(PTS_RS = `Regular Season`,
         PTS_PO = Playoffs)

lebron_reb_diff <- lebron_reb_diff |>
  rename(REB_RS = `Regular Season`,
         REB_PO = Playoffs)

lebron_ast_diff <- lebron_ast_diff |>
  rename(AST_RS = `Regular Season`,
         AST_PO = Playoffs)

lebron_diff <- lebron_pts_diff |>
  full_join(lebron_reb_diff, by = "Season")|>
  full_join(lebron_ast_diff, by = "Season")

# Adding PRA line

lebron_diff$pra_rs <- lebron_diff$PTS_RS + lebron_diff$REB_RS + lebron_diff$AST_RS
lebron_diff$pra_po <- lebron_diff$PTS_PO + lebron_diff$REB_PO + lebron_diff$AST_PO
lebron_diff$pra_difference <- lebron_diff$pra_po - lebron_diff$pra_rs

Average Difference By Metric

pts_diff <- mean(lebron_diff$pts_difference)
reb_diff <-mean(lebron_reb_diff$reb_difference)
ast_diff <-mean(lebron_diff$ast_difference)
pra_diff <- mean(lebron_diff$pra_difference)

On average, LeBron performed better in the playoffs, scoring 1.07 more points and grabbing 1.24 more rebounds. However, his assists decreased slightly by 0.33, while overall production (PRA) increased by 1.99.

Statistical Testing

To test whether these observed differences were statistically significant, I used paired t-tests comparing LeBron’s regular season and playoff performance across seasons.

Points Paired T-test

# Running Paired T-Tests
# Pts T-Test

# Histogram of Differences
hist(lebron_diff$pts_difference,
     main = "Differences in Regular Season and Playoff PPG",
     xlab = "Difference in PPG")

pts_ttest <- t.test(lebron_diff$pts_difference, mu = 0)

Points Interpretation:

On average, LeBron James scores approximately 1.07 more points per game in the playoffs compared to the regular season. However, a paired t-test indicates that this difference is not statistically significant (t = 1.34, p = 0.198), indicating the increase is not consistent across seasons.

Rebounds Paired T-test

hist(lebron_diff$reb_difference,
     main = "Differences in Regular Season and Playoff Rebounds",
     xlab = "Difference in REB")

reb_ttest <- t.test(lebron_diff$reb_difference, mu = 0)

Rebounds Interpretation:

In contrast, LeBron averages 1.24 more rebounds per game in the playoffs than in the regular season. This difference is statistically significant (t = 3.83, p = 0.001), indicating a consistent increase in rebounding performance during postseason play.

Assists Paired T-test

hist(lebron_diff$ast_difference,
     main = "Differences in Regular Season and Playoff Assists",
     xlab = "Difference in AST")

ast_ttest <- t.test(lebron_diff$ast_difference, mu = 0)

Assists Interpretation:

LeBron averages 0.33 fewer assists per game in the playoffs compared to the regular season. This difference is not statistically significant (t = -1.28, p = 0.217), suggesting no consistent change in playmaking output.

PRA Paired T-test

hist(lebron_diff$pra_difference,
     main = "Differences in Regular Season and Playoff Points, Rebounds, and Assists",
     xlab = "Difference in PRA")

pra_ttest <- t.test(lebron_diff$pra_difference, mu = 0)

Points, Rebounds, Assists (PRA) Interpretation:

Overall, LeBron averages 1.99 more combined points, rebounds, and assists (PRA) in the playoffs. However, this difference is not statistically significant (t = 2.03, p = 0.0587), barely missing the conventional significance threshold of α = 0.05.

Key Findings

Overall, the analysis shows that LeBron’s performance remains largely consistent between the regular season and playoffs. While scoring, assists, and overall PRA do not show statistically significant changes, rebounding demonstrates a clear and consistent increase in the postseason. This suggests that while LeBron maintains a high level of performance year-round, he contributes additional impact in rebounding during playoff competition.

  • Scoring (PPG): Increased slightly in the playoffs, but not statistically significant.
  • Rebounding (RPG): Increased in the playoffs and was statistically significant, indicating a consistent boost in rebounding impact.
  • Assists (APG): Decreased slightly in the playoffs, but not statistically significant.
  • Overall Offensive Production (PRA): Increased modestly, but did not reach statistical significance.

Limitations

While this analysis provides useful insights, several limitations should be considered:

  • Dataset required manual cleaning and adjustments, which may introduce mionr inconsistencies.
  • The analysis uses a relatively small sample size (~20 Seasons), limiting statistical power.
  • Only used basic box score metrics (points, rebounds, assists)
  • Contextual factors such as team composition, opponent strength, and role changes were not considered
